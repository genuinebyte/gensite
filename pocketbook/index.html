Light Rust

Curated list of pretty good rust crates with a minimal amount of dependencies.

MiniHttps

Rust crate for serving content over HTTPS with the least number of dependencies I can manage. No fancy features.

Pixie

Simplistic pixel art program written in Rust. For fun.

Essential Fonts

AUR package that covers as much of Unicode as possible with the least amount of fonts.

init.rs

Linux init system written in Rust. Should be safe and respectful of PID 1.

Breaking

Rust crate for interacting with the NewsAPI. Possibly other news outlets too.

linode-rs

Rust crate for interacting with the Linode API.

CloudBacked

Program to encrypt chunks of data and store them on various file hosts.
